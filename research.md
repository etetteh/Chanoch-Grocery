# UX/UI Research: Toronto Sale Scout

## 1. Modern Grocery App UX Best Practices (2025)
- **Smart Grouping**: Items should be grouped by store or aisle to minimize walking distance in-store.
- **Price History/Trends**: Show if a "sale" is actually a good deal compared to historical prices.
- **Collaborative Lists**: Allow users to share lists with family/roommates (Real-time sync).
- **Location-Based Alerts**: Notify users when they are near a store with a saved deal.
- **Visual Scannability**: Use high-contrast badges for "Best Price", "Organic", "Local", etc.
- **One-Tap Actions**: Minimize clicks to add/remove items.

## 2. Toronto-Specific Pain Points
- **Store Proximity**: Toronto has many "food deserts" and dense areas where store choice is limited by transit.
- **Flyer Fragmentation**: Sales are spread across Flipp, Reebee, and individual store apps (Loblaws, Metro, etc.).
- **Price Volatility**: Prices for produce (especially berries and leafy greens) fluctuate wildly in Ontario.
- **Store Chains**: Understanding which stores are "discount" (No Frills, FreshCo, Food Basics) vs "premium" (Loblaws, Metro, Sobeys).

## 3. UI Improvement Opportunities
- **Maps Integration**: Show the exact location of the store on a map to help users decide if the "Best Price" is worth the travel.
- **Bento Grid Layout**: For popular categories to make the home screen more engaging.
- **Micro-animations**: For adding items to the list (visual confirmation).
- **Dark Mode Support**: Essential for late-night list making.
- **Accessibility**: Larger touch targets for in-store use (one-handed operation).

## 4. Proposed Enhancements for Toronto Sale Scout
1.  **Maps Grounding**: Integrate `googleMaps` tool to show store locations and provide directions.
2.  **Smart Sorting**: Sort the shopping list by store to make the actual shopping trip more efficient.
3.  **Visual Comparison**: Use a "Price Ladder" UI for better comparison.
4.  **Budget Estimate**: Show the total estimated cost of the shopping list.
5.  **Store Logos**: Add visual branding for major Toronto retailers.
